# 🚀 Descripción

**BarberStyle** es una landing page para una barbería desarrollada con **React 19**, **Vite** y **Bootstrap**.

Incluye:

- ✅ Navbar Sticky
- ✅ Tarjetas de servicios
- ✅ Modales
- ✅ Galería con filtros
- ✅ Lightbox
- ✅ Sistema de reservas
- ✅ CRUD con LocalStorage
- ✅ API externa (randomuser.me)
- ✅ Diseño Responsive
